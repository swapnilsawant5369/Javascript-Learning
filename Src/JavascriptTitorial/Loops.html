While Loop

Loops are handy, if you want to run the same code over and over again, each time with a different value.

var a = 1;
while (a <= 15) {
document.write(a + ") " + "Hello Good Morning Everyone.");
a++;
}


Do while Loop

do {
document.write(a + ") " + "Hello Everyone.");
a++;
} while (a <= 20);


For Loop

for (var sam = 1; sam <= 30; sam++) {
document.write(sam + " )" + "Welcome City.");
}
